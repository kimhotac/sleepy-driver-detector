졸음 판정 알고리즘
(Time-based Drowsiness Detection)

👁️
1. 눈 감김 감지
양쪽 눈이 감겼는지(1) 확인
처음 감지 시 시작 시간 기록
→
⏱️
2. 시간 누적
계속 감긴 상태면
현재 시간 - 시작 시간을 누적
눈 뜨면 누적 시간 초기화
→
🚨
3. 졸음 판정
누적 시간 ≥ 1초(1000ms)면
졸음 상태로 판정
💡 프레임 수가 아닌, 실제 시간(ms) 기준으로
눈 감김의 지속 여부를 판단합니다.

왜 시간 기반 판정을 선택했는가?

⚠️
프레임 기반의 한계
웹캠의 프레임 속도는 환경에 따라
불규칙하게 변할 수 있습니다.
(예: 1초에 30프레임이 아닐 수도 있음)
⏳
시간 기반의 장점
각 프레임의 실제 촬영 시각을 이용해
정확한 눈 감김 지속 시간을 측정할 수 있습니다.
🚗
1초 기준의 근거
시속 100km로 1초만 졸아도
차량은 약 28m를 질주합니다.
이는 음주운전보다 치사율이 높으며,
1초 이상 눈을 감는 것은 매우 위험합니다.
→ 1초(1000ms) 이상 감기면 졸음 판정
